# ML Classification Model Comparison
This repository contains a machine learning classification assignment focused on comparing different models and understanding how their assumptions affect performance.
The project evaluates KNN, Support Vector Machines (SVM), and Random Forest using several experiments including hyperparameter tuning, probability analysis, and robustness testing.

# Methods
Train/test split with stratification
Feature scaling using StandardScaler
Majority-class baseline model
KNN classification with threshold analysis
Linear and RBF SVM with GridSearchCV
Random Forest hyperparameter tuning

# Experiments
Feature rescaling to test model sensitivity
Comparison of decision scores vs probabilities
Distribution shift simulation on the test set

# Tools
Python, NumPy, Pandas, Scikit-learn, Matplotlib, Seaborn

# Goal
The aim of this project is to understand how different machine learning models behave under varying conditions, rather than focusing only on maximizing accuracy.
